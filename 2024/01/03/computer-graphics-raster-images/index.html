Image Storage

To reduce the storage requirement, most image formats allow for some kind of compression. Such compression is either lossless or lossy. Popular image storage formats include

jepg: lossy format compresses image blocks on thresholds in the human visual system.

tiff: lossless, most commonly used to hold binary images

ppm: losslessd, uncompressed format

png: a set of lossless formats
